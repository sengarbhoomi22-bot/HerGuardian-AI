# HerGuardian AI

A wellness platform for women focusing on health, nutrition, mental well-being, and community support.

## Structure
- `frontend/` - React + Vite application
- `backend/` - Node.js + Express API

## Deployment
- Configure environment variables in `backend/.env`
- Start backend with `npm --prefix backend run dev`
- Start frontend with `npm --prefix frontend run dev`

## Notes
- Removed legacy AI Assistant, Symptom Checker, Period Tracker, and Wellness Insights modules.
- Root `.gitignore` now ignores Node modules and local environment files.
