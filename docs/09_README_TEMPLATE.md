# 🩷 HerGuardian AI

> **Empowering Women Through AI, Knowledge, Safety & Growth**

---

## 📌 Overview

HerGuardian AI is an AI-powered women empowerment platform built using the **OpenAI Agents SDK**.

Instead of requiring women to search across multiple websites, HerGuardian AI provides one intelligent platform for trusted guidance related to:

- Women's Health
- Vegetarian Nutrition
- Women's Fitness
- Mental Wellness
- Menstrual Wellness
- Women's Safety
- Career Guidance
- Scholarships
- Inspiration
- AI Chat Assistance

The platform is designed to be globally accessible, multilingual, user-friendly, and professionally engineered.

---

# 🌍 Problem Statement

Women often need reliable information regarding health, safety, education, careers, nutrition, and wellbeing.

However,

- Information is scattered
- Sources are difficult to verify
- Personalization is limited
- Multiple websites are required

HerGuardian AI solves this by acting as one trusted digital companion.

---

# 💡 Solution

HerGuardian AI combines AI, trusted educational resources, multilingual accessibility, and modern software architecture into one unified platform.

The platform helps users discover trusted information, learn through articles and videos, save resources, and interact with an intelligent AI assistant.

# 🚀 Installation

Clone the repository

```
git clone <repository-url>
```

Install frontend

```
npm install
```

Install backend

```bash
cd backend
npm install
```

Run frontend

```bash
cd frontend
npm run dev
```

Run backend

```bash
cd backend
npm run dev
```

---

# 🔑 Environment Variables

Create:

```
backend/.env
```

Example

```
OPENAI_API_KEY=your_api_key_here
```

Never commit the `.env` file.
